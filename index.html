Getting Started

1

Enable Screen Time Access

When you first open Potato Mode, tap Enable Screen Time Access. iOS will ask you to confirm — this is required to block apps. No data leaves your device.

2

Pick your apps

Tap the app picker to choose which apps you want to block. You can select individual apps or entire categories like Social or Entertainment.

3

Block your way

Use Couch Mode for instant blocking, Schedules for recurring time windows, or Usage Limits to set a daily time budget per app group.

Frequently Asked Questions

Why does Potato Mode need Screen Time access?
Potato Mode uses Apple's Screen Time API to apply blocks — it's the same system your iPhone uses for built-in parental controls and focus modes. This is the only way to block apps on iOS. Your data never leaves your device and Potato Mode has no servers or accounts.
An app is blocked and I can't turn it off
Open the Potato Mode app and check each section:

• Home tab — make sure Couch Mode is toggled off
• Schedules tab — check if an active schedule is running
• Limits tab — a usage limit may have triggered for the day

Toggling the relevant switch off should remove the block immediately.
My usage limit triggered but I barely used the app
This is a known issue with iOS's Screen Time API where a threshold can occasionally fire incorrectly — particularly when the device is plugged in while idle. Apple is aware and investigating.

Workaround: Simply open Potato Mode. The app automatically detects and clears any phantom blocks when you launch it.
Will my blocks survive if I delete and reinstall the app?
Deleting Potato Mode will remove all blocks and reset Screen Time permissions. Your phone will return to its normal state. If you reinstall, you'll need to re-grant Screen Time access and set up your blocks again.
Can I block all apps or system apps like Safari?
You can block most third-party apps and many Apple apps including Safari. Some core system apps (like Settings and the Phone app) cannot be blocked by Apple's design — this is a platform-level restriction to ensure you can always access emergency functions.
Does Potato Mode work if my phone is off or in airplane mode?
Schedules and usage limits are managed by iOS in the background, so they continue to work regardless of network connection. Potato Mode doesn't require internet access at any point.
How do I remove Potato Mode completely?
Simply delete the app from your home screen like any other app. All blocks will be removed automatically. You may also want to go to Settings → Screen Time to confirm everything has been cleared.
